inactivar_usuario(request, pk_usuario)

source code 

Funcion que inactiva la cuenta de un usuario seleccionado.

Parameters:
  • request (django.http.HttpRequest) - Contiene informacion sobre la peticion actual
  • pk_usuario (string) - id del usuario a ser inactivado
Returns: django.http.HttpResponseRedirect
Renderiza usuarios/delete.html para obtener el formulario o redirecciona a la vista index de usuarios si el usuario fue desactivado.
Decorators:
  • @login_required(login_url= '/login/')
  • @permission_required('usuarios.inactivar_usuario')
